Binary/Multi-Class Classification

This project explores two key approaches in machine learning classification, binary and multi-class classification. Through interactive visualizations, users can plot points on a canvas to dynamically classify data points based on their chosen category.

About The Project

This project is an interactive demonstration of binary and multi-class classification techniques. It allows users to visualize how machine learning algorithms work by interacting directly with the canvas, placing data points, and seeing how the model learns to distinguish between different categories.

Binary Classification

Binary classification predicts one of two possible classes. For example:

Example Input Prediction
Email Content Spam / Not Spam
Image Cat / Dog

Multi-Class Classification

Multi-class classification assigns data points to one of multiple categories. For example:

Example Input Prediction
Handwritten Digit 0 / 1 / 2 / ... / 9
Animal Image Cat / Dog / Bird
